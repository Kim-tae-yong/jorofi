This is JoRoFi, a Jogging Route Finder.
Please have a look at the original website:
https://code.google.com/p/jorofi/

You may fork this code, embed it in commercial software.


INSTALL:
========

Build
- cd base
- mvn install
- cd ..

Release
- cd release
- mvn assembly:assembly
- cd target
- now you have jorofi-jar-with-dependencies.jar

Run
- java -jar jorofi-jar-with-dependencies.jar
- follow help


For any help, look at https://code.google.com/p/jorofi/ or contact dutoitc@gmail.com
